## Overview
Basic button component. Allows users to trigger actions by interacting with the button by mouse or keyboard.
Comes with different variations depending on the type of action we want to emphasise to the user.

<Story
  name="Button"
  parameters={{
    layout: "centered",
    jest: ["ui/button/button"],
  }}
>
  <Button tooltip="This button has a tooltip.">
    Click me!
  </Button>
</Story>

Can be imported from:

```jsx
import { Button } from "@digital-science/figshare-fcl/button";

<Button 
  theme="primary" 
  onClick={() => {}} 
  tooltip="This button has a tooltip"
>
  Main action
</Button>
<Button 
  theme="secondary" 
  onClick={() => {}} 
  tooltip="This button has a tooltip"
>
  Secondary action
</Button>
```
