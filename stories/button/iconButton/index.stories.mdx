## Overview

Similar to a Button component, but for contexts where we want to represent actions as visual Icons instead of Text labeled buttons.
Useful for when we have limited space or a label would be redundant and an `Icon` would more clearly, and concicely, represent the action.
Comes with different variations depending on the type of action we want to emphasise to the user.

<Story
  name="Icon Button"
  parameters={{
    layout: "centered",
    jest: ["ui/src/button/iconButton"],
  }}
>
  <IconButton Icon={Home}>
    This IconButton has a tooltip.
  </IconButton>
</Story>

Can be imported from:

```jsx
import React from "react";
import Home from "@figshare/fcl/icons/home";
import { IconButton } from "@figshare/fcl/button";

class MyButton extends React.Component {
  render() {
    return (
      <IconButton Icon={Home}>
        This IconButton has a tooltip, 
        and these are the contents of that tooltip.
      </IconButton>
    );
  }
}
```
