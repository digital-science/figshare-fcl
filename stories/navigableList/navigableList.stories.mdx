## Overview

NavigableList component, used to render breadcrumbs or inline, ordered link buttons.
Use *Left and Right Arrow* keys to navigate between focusable elements for `"horizontal"` `navigationType`s, 
or *Up and Down Arrow* keys for `"vertical"` `navigationType`s.

The component can be imported from:

```jsx
import NavigableList from "@digital-science/figshare-fcl/helpers/navigableList";

const containerStyle = {
  margin: "calc(3 * var(--gridSize))",
  padding: "calc(3 * var(--gridSize))",
  display: "flex",
  flexWrap: "wrap",
};

const buttonStyle = {
  margin: "calc(2 * var(--gridSize))",
};

const renderContent = ({ ref, ...passedProps }) => (
  <div ref={ref} style={containerStyle} {...passedProps}>
    <button style={buttonStyle}>Item 1</button>
    <button style={buttonStyle}>Item 2</button>
    <button style={buttonStyle}>Item 3</button>
  </div>
);

function ExampleHorizontalNavigableList() {
    return (
      <>
        <div style={containerStyle}>
          <NavigableList navigationType="horizontal">
            {renderContent}
          </NavigableList>
        </div>
        <span></span>
      </>
    );
}
```
