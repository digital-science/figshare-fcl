## Overview

New button component, based on `data-*` attributes instead of CSS classes.
Supports `kind`, `em` (emphasis) variants, loading state, tooltip and rendering as a link.

<Story
  name="Button"
  parameters={{ layout: "centered" }}
>
  <Button onClick={() => {}}>
    <Button.Label>Click me</Button.Label>
  </Button>
</Story>

Poate fi importat din:

```jsx
import { Button } from "@digital-science/figshare-fcl/newButton";

<Button kind="primary" em="high" onClick={() => {}}>
  <Button.Label>Label</Button.Label>
</Button>
```
