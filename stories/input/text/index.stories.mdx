## Overview

A text input component. Allows users to type in and store text values.

<Story 
  name="Text" 
  parameters={{
    layout: "centered",
    jest: ["ui/src/input"],
  }}
>
  <TextInput placeholder="Input text..." onChange={(event) => {}} />
</Story>

Can be imported directly or referenced as the default `Input` component variation:

```jsx
import React from "react";
import Input from "@figshare/fcl/input";
import TextInput from "@figshare/fcl/input/text";

function TextInputs() {
  return (
    <TextInput onChange={(event) => {}} />
    <Input onChange={(event) => {}} />
    <Input type="text" onChange={(event) => {}} />
  );
}
```
