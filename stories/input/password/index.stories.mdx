## Overview
Password input component. Allows users to safely input passwords, and to temporarily toggle visibility.

<Story 
  name="Password" 
  parameters={{
    layout: "centered",
    jest: ["ui/src/input"],
  }}
>
  <PasswordInput onChange={(event) => {}}/>
</Story>

Can be imported directly or referenced as an `Input` component variation, by setting `type="password"`:

```jsx
import React from "react";
import Input from "@figshare/fcl/input";
import PasswordInput from "@figshare/fcl/input/password";

function PasswordInputs() {
  return (
    <PasswordInput onChange={(event) => {}} />
    <Input type="password" onChange={(event) => {}} />
  );
}
```
